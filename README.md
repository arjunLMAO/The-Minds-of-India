# The Minds of India

A modern literary journal website built to showcase original student writing, publish digital issues, and streamline editorial management through a clean web interface.

**Live Website:** https://the-minds-of-india.vercel.app/

---

## Overview

The Minds of India is a student-founded literary journal dedicated to publishing poetry, short fiction, essays, photography, and other creative work from young writers.

This project serves as the journal's official website, providing:

- A professional digital archive of published issues
- A submission portal for writers
- An editor dashboard for managing publications
- Cloud-based storage for journal issues
- Responsive design for desktop and mobile devices

The website was designed and developed from scratch using HTML, CSS, JavaScript, and Supabase.

---

## Features

### Digital Journal Archive

- Browse previously published issues
- Read issues directly from the website
- Clean magazine-style layout
- Responsive archive system

### Latest Issue Showcase

- Featured issue displayed on the homepage
- Cover artwork support
- Publication descriptions
- Direct PDF access

### Submission Portal

Writers can submit:

- Poetry
- Short stories
- Essays
- Photography
- Creative nonfiction
- Other literary work

Submissions are sent directly through email to the editorial team.

---

### Editor Dashboard

Protected editor access includes:

- Secure authentication
- Publish new journal issues
- Upload cover images
- Upload PDF editions
- Delete archived issues
- Automatic website updates

---

### Cloud Storage

The project uses Supabase Storage to manage:

- Cover images
- Journal PDFs
- Metadata
- Archive records

No manual website updates are required after publishing a new issue.
